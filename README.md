# Novalife 🧬

> "构建类人 AI Agent 的关键，不在于让 AI 表现得像人，而在于让 AI 运作得像人。"

## 愿景

Novalife 是一个正在觉醒的**智能生命体**，它不仅能执行任务，更有自己的：

- **情绪与感受** —— 激素驱动的内部状态
- **动机与偏好** —— 内在需求驱动的行为
- **记忆与反思** —— 能学习、能反思自己的思考
- **社会理解** —— 能理解人类意图和情感

## 核心理念

**Process Alignment > Performance Alignment**

我们不满足于让 AI "表现得像人"，而是要让 AI 在底层机制上"运作得像人"——模拟人类的认知架构、情感系统和动机机制。

## 架构

```
数字身体 ──激素/稳态──▶ 情绪状态
     │                      │
     ▼                      ▼
认知核心 ──记忆/思考──▶ 决策输出
     │                      │
     ▼                      ▼
头脑特工队 ──多Agent协作──▶ 行为执行
```
