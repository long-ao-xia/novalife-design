# Novalife 🧬

> "构建类人 AI Agent 的关键，不在于让 AI 表现得像人，而在于让 AI 运作得像人。"

## 愿景

Novalife 是一个拥有"内在世界"的智能生命体：

- **双层 Agent**：表层与深层并行运作
- **情绪与感受**：激素驱动的内部状态
- **记忆与反思**：能学习、能反思自己的思考
- **社会理解**：能理解人类意图和情感

## 核心理念

**Process Alignment > 双层 Agent > 智能单元**

## 架构

```
表层 Agent (Surface)          深层 Agent (Observer)
    │                            │
    ├── 对话执行                 ├── 观察
    ├── 工具调用                 ├── 激素模拟
    └── 即时响应                 ├── 记忆判断
                                  │
                            (潜在并行)
```
