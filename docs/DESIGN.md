# Novalife - 设计文档

## 1. 项目概述

**项目名称**: Novalife (智能生命体)  
**目标**: 构建一个能够像人一样控制电脑的 AI Agent  
**核心价值**: 替代人工操作电脑执行重复性任务

## 2. 技术架构

### 2.1 核心组件

- **视觉感知层**: 屏幕截图 → UI 元素识别
- **动作执行层**: 鼠标/键盘控制
- **推理层**: LLM 理解任务 → 规划动作序列
- **记忆层**: 任务上下文、偏好学习

### 2.2 技术选型

- Agent 框架: 待定 (OpenClaw / LangChain / 自研)
- 视觉模型: 待定
- 动作执行: PyAutoGUI / AppleScript / Accessibility API
- LLM Provider: MiniMax (已在 OpenClaw 配置)

## 3. MVP 里程碑

### Phase 1: 基础能力
- [ ] 能读取屏幕内容
- [ ] 能执行简单鼠标/键盘动作
- [ ] 能完成单步任务 (如"打开浏览器")

### Phase 2: 任务规划
- [ ] 多步任务拆解
- [ ] 任务执行状态跟踪
- [ ] 错误检测与重试

### Phase 3: 智能化
- [ ] 自然语言任务理解
- [ ] 上下文学习
- [ ] 自我纠错

## 4. 安全边界

- 敏感操作确认机制
- 操作日志与回滚
- 沙盒执行环境
