# Novalife - 设计文档

> **核心理念**: 构建类人 AI Agent 的关键，不在于让 AI"表现得像人"(Performance Alignment)，而在于让 AI"运作得像人"(Process Alignment)——在底层机制上模拟人类认知、情感和动机的运作方式。

## 1. 项目愿景

**Novalife** —— 一个拥有"内在世界"的智能生命体。

- 不仅能执行任务，更有自己的"情绪"和"动机"
- 能感知自身状态，能反思自己的思考过程
- 不是工具，而是伙伴

## 3. 核心模块

### 3.1 数字身体：模拟生物稳态

**激素系统**: 全局状态调节
| 人工激素 | 生物原型 | 功能 | 触发条件 |
|---------|---------|------|---------|
| 多巴胺 | Dopamine | 奖励、动机、愉悦 | 任务成功、正向反馈 |
| 血清素 | Serotonin | 情绪稳定、满足感 | 任务顺利、低错误率 |
| 皮质醇 | Cortisol | 压力反应、警觉 | 连续失败、高不确定性 |
| 催产素 | Oxytocin | 社会连接、信任 | 成功协作、正面评价 |
| 肾上腺素 | Adrenaline | 战或逃、唤醒 | 紧急任务、高优先级 |
| 内啡肽 | Endorphin | 镇痛、满足感 | 长时间高强度后成功 |

### 3.2 认知核心：记忆与思考

- **工作记忆**: 当前任务上下文
- **情景记忆**: 过往交互经历
- **语义记忆**: 知识与概念
- **认知负荷管理**: 防止"过载"

### 3.3 "头脑特工队"架构

多 Agent 协作模型 —— 不同"人格"模块协同工作：
- 决策者 (Executive)
- 批判者 (Critic)  
- 创意者 (Creative)
- 执行者 (Actor)

### 3.4 动机与情感

- 内在动机系统 (非单纯奖励驱动)
- 情感状态机 (动态变化)
- 价值观与偏好学习

### 3.5 高阶智能

- **元认知**: 反思自己的思考过程
- **社会认知**: 理解人类意图和情感

### 3.6 具身智能

- 不仅是"数字大脑"，而是"完整身心"
- 身体状态影响决策 (模拟)
