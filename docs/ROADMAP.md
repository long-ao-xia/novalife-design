## 智能单元详解 (M3)

**定义**：激素系统与记忆系统共同构成智能单元，单独存在无意义。

### 激素 × 记忆

| 激素 | 对记忆的影响 |
|------|-------------|
| 多巴胺 | 愉悦时刻加深记忆 |
| 皮质醇 | 压力时刻强化记忆 |
| 血清素 | 稳定状态正常存储 |
| 催产素 | 重要关系加深记忆 |

### 深层 Agent 职责

1. **观察**：监听表层与用户交互
2. **模拟**：根据观察结果模拟激素变化
3. **判断**：识别值得加深记忆的点
4. **影响**：激素状态可反馈至表层行为
