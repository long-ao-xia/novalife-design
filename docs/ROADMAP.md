# Novalife 路线图 (Roadmap)

> 版本: v1.0
> 状态: 实现智能化的里程碑路线

---

## 核心理念

**Process Alignment > 单一心智 > 简单优先**

目标：构建一个"像人一样运作"的 AI Agent

---

## 里程碑

### M0: 基础骨架

> 当前阶段

- [x] 项目初始化 (TypeScript)
- [x] Tape 存储 (JSONL)
- [x] LLM 调用 (MiniMax)
- [x] 基础工具 (echo, memory)
- [x] Telegram Bot 接口

**交付物**: 能在 Telegram 对话的简单 Agent

---

### M1: 自举能力

> 目标：能自己改进自己

- [ ] 4 工具集 (read, write, edit, bash)
- [ ] 极简 system prompt (< 1000 tokens)
- [ ] Agent Loop (循环直到完成)
- [ ] 反思写入文件
- [ ] 验证机制 (测试运行)

**关键问题**: 机器能自己改进自己吗？

---

### M2: 激素系统

> 目标：像人一样有"情绪"

- [ ] 信心激素 (confidence)
- [ ] 好奇激素 (curiosity)
- [ ] 压力激素 (stress)
- [ ] 状态影响决策

**关键问题**: 情绪如何影响行为？

---

### M3: 记忆分层

> 目标：像人一样遗忘

- [ ] Tape (完整历史)
- [ ] Working (当前上下文)
- [ ] Long-term (重要记忆)
- [ ] 遗忘机制

**关键问题**: 什么该记住？什么该忘记？

---

### M4: 多模态感知

> 目标：像人一样感知世界

- [ ] 视觉输入
- [ ] 声音输入
- [ ] 持续注意力

**关键问题**: 有限注意力如何分配？

---

### M5: 自主意图

> 目标：像人一样有"自己的想法"

- [ ] 内部动机生成
- [ ] 自我提问
- [ ] 主动探索

**关键问题**: 机器能有自己的目标吗？

---

## 里程碑关系

```
M0 ───── M1 ───── M2 ───── M3 ───── M4 ───── M5
 基础    自举    激素    记忆    感知    意图
```

每个里程碑建立在前一个之上。

---

## 成功标准

| 里程碑 | 验证方式 |
|--------|---------|
| M0 | Telegram 对话可用 |
| M1 | 能自己修复 bug |
| M2 | 失败后信心下降 |
| M3 | 记得重要的，忘记琐碎的 |
| M4 | 多模态输入理解 |
| M5 | 主动提出问题 |
