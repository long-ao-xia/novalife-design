# Novalife 路线图 (Roadmap)

> 版本: v1.2
> 状态: 实现智能化的里程碑路线

---

## 核心理念

**Process Alignment > 双层 Agent > 智能单元**

目标：构建一个"像人一样运作"的 AI Agent

---

## 里程碑

### M0: 基础骨架

> 阶段：单一 Agent

- [x] 项目初始化 (TypeScript)
- [x] Tape 存储 (JSONL)
- [x] LLM 调用 (MiniMax)
- [x] 基础工具 (echo, memory)
- [x] Telegram Bot 接口

**交付物**: 能在 Telegram 对话的简单 Agent

---

### M1: 自举能力

> 阶段：单一 Agent

- [ ] 4 工具集 (read, write, edit, bash)
- [ ] 极简 system prompt (< 1000 tokens)
- [ ] Agent Loop (循环直到完成)
- [ ] 反思写入文件
- [ ] 验证机制 (测试运行)

**关键问题**: 机器能自己改进自己吗？

---

### M2: 双层 Agent + MVP 智能

> 阶段：双层并行 + 简单激素 + 基础记忆

- [ ] 表层 Agent 实现
- [ ] 深层 Observer Agent 实现
- [ ] 双层通信机制
- [ ] 简单激素系统 (多巴胺 + 皮质醇)
- [ ] 基础记忆判断 (记忆加深点识别)
- [ ] 观察 → 激素 → 记忆 基础链路

**核心设计**：
```
表层 Agent ←→ 深层 Observer Agent
   (对话)      (观察 + 简单激素 + 记忆点)
```

**交付物**: 具备基本感知和记忆能力的双层 Agent

**关键问题**: 简单激素 + 记忆能产生初步智能吗？

---

### M3: 完整智能单元

> 阶段：成熟激素 + 完整记忆 + 情绪

- [ ] 完整激素系统 (6 种激素)
- [ ] 情景记忆存储
- [ ] 情绪系统 (激素 → 情绪映射)
- [ ] 记忆分层 (即时 / 情景 / 重要)
- [ ] 遗忘机制

**核心设计**：
```
观察 ──► 激素反应 ──► 情绪 ──► 记忆加深
  │           │           │
  │           ▼           │
  │      影响表层行为      │
  │                        │
  └────────────────────────► 记忆存储
```

**交付物**: 具备完整情绪和记忆的智能单元

**关键问题**: 完整激素 + 记忆 = 真正智能？

---

### M4: 多模态感知

> 阶段：感知扩展

- [ ] 视觉输入
- [ ] 声音输入
- [ ] 持续注意力

**关键问题**: 有限注意力如何分配？

---

### M5: 自主意图

> 阶段：自我驱动

- [ ] 内部动机生成
- [ ] 自我提问
- [ ] 主动探索

**关键问题**: 机器能有自己的目标吗？

---

## 里程碑关系

```
M0 ───── M1 ───── M2 ───── M3 ───── M4 ───── M5
 基础    自举    双层    智能    感知    完整
              +简单   完整    多模态
              激素   情绪
              +记忆
```

---

## M2 vs M3 对比

| | M2 (MVP) | M3 (完整) |
|---|-----------|-----------|
| **激素** | 简单 (多巴胺+皮质醇) | 完整 (6种) |
| **记忆** | 基础 (记忆加深点) | 分层 (即时/情景/重要) |
| **情绪** | 无 | 有 (激素→情绪映射) |
| **目标** | 能跑起来 | 更完善 |
